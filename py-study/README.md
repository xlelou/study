## 笔记
变量名：可以包括英文、数字以及下划线，但不能以数字开头，区分大小写

变量类型：弱类型语言、无需声明
数字Number：整型和浮点型
字符串String：字符串拼接、长度、切片
列表List：添加元素、求长、切片、删除
元组Tuple：readonly
字典Dictionary：赋值、判断是否存在某个key

注释：#，三引号

保留字符：and，not，class，def，等等等等 

行和缩进

运算符：
算术运算符：+，-，* ，/，%
比较运算符：==，!=，>，<，>=，<=
赋值运算符：=，+=，-=，*=，/=，%=
逻辑运算符：and，or，not

条件：
if...
if…else...
if…elif…else

循环：
while
for，for遍历list和dict

循环控制：
break
continue
pass

时间：time.time()

文件：读写文件

异常

函数：def
